구조 설계

0. 폴더 구조
core - 코어, 핵심 컴포넌트들
apps - 페이지 동작 구성
pages - 페이지 html
styles - 페이지 css
static - 페이지의 이미지와 같은 정적 파일
routers - 페이지 네비게이터

app.js - 모든 연결구심점
index.html - 진입점

1. 기본적으로 Component는 다음과 같다.
생성
템플릿
렌더
이벤트설정
렌더링 순서

생성에는 해당 객체에 대한 정보를
템플릿은 해당 객체가 보여줄 html을
이벤트는 버튼과 같은 이벤트 설정을,
렌더링 순서는 순서를 지정한다.

다음과 같이 설정하면 편하다.
1) 생성자에서 html 파일 로드
2) template에 로드한 파일 입력
3) 이때, css를 입력한 뒤, 넣는다
4) 각 요소에 대한 이벤트 설정
